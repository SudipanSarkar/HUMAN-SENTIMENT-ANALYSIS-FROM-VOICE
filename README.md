**SPEECH EMOTION RECOGNITION USING SPECTOGRAM FEATURES AND CNN UNDER RESOURCE CONSTRAINED ENVIRONMENT**
Speech-based sentiment analysis focuses on identifying human emotions from voice signals. Unlike text-based approaches, speech carries additional information such as tone,pitch, and intensity, which helps in understanding emotions more effectively. In this project, we have designed and implemented a sentiment analysis system using speech data collected from publicly available datasets. The system processes audio signals in both time and frequency domains. Waveforms and log power spectrograms are used as input features for the model. Since many existing models require high computational power, our work focuses on building a lightweight solution that can operate in resourceconstrained environments such as IoT devices and embedded systems. We combined multiple datasets including SAVEE, RAVDESS, TESS, and CREMA-D, resulting in a total of 11,362 audio samples across different emotion classes. We trained the model using audio samples at 8 kHz and 16 kHz. After testing, we found that the system gives around persentage 78 accuracy and still works efficiently.We also designed the system so that it can run on an ESP32 with a microphone for real-time emotion detection. 
Keywords : Speech Emotion Recognition, Voice-based Sentiment analysis, Spectrogram,Lightweight Model, Embedded Systems, ESP32, CNN, Audio Processing.
